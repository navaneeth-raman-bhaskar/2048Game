# 2048 Game

A PHP command line game built from scratch.

Supported version  `PHP 7.4`, `PHP 8.0` and `PHP 8.1`

## Want to try?

- Fork or clone the repo
- Then run `php run.php`
- Enjoy!

## Releases
- `v1.0.0` for *PHP 8* or greater
- `v1.0.0-beta` for *PHP 7.4*
